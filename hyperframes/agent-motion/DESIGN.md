# StratEx Agent Motion Visual Identity

## Style Prompt

Dark DeFi command-center animation with a premium StratEx-inspired astronaut/vault aesthetic. The frame should feel secure, technical, and operational: glowing route lines, contract nodes, capital packets, and an agent core coordinating Base and Arbitrum deployments.

## Colors

- Background: `#020814`
- Panel navy: `#071526`
- Electric cyan: `#4ee8ff`
- Chain blue: `#2ea8ff`
- Safe mint: `#79f2c0`
- Alert amber: `#ffd166`
- Text white: `#f8fbff`

## Typography

- Display: Orbitron, Rajdhani, Eurostile, system UI fallback
- Labels: Inter, Sora, Avenir Next, system UI fallback

## What NOT to Do

- Do not use purple or orange as the dominant palette.
- Do not include readable fake brand text beyond the deliberate labels.
- Do not use randomized motion or infinite repeat timelines.
- Do not make the frame look like a marketing landing page.
